Ex 5QROC
Le reflux hépato-jugulaire est pathognomonique de l'insuffisance cardiaque droite (ce signe ne se voit que dans cette pathologie). En règle logique, qu'est-ce que cela signifie en termes de Se, Sp, VP, VN, FP, FN ?
Réponse type (corrigé officiel) :

Si le patient n'a pas d'IC droite, il n'a pas de reflux hépato-jugulaire → il n'existe pas de faux positifs (FP = 0) → la spécificité est de 100%. Présence du signe ⇒ diagnostic certain (VPP = 100%).

Ex 6QROC
L'élévation des transaminases est permanente dans l'insuffisance hépatique. En règle logique, qu'est-ce que cela signifie en termes de Se, Sp, VP, VN, FP, FN ?
Réponse type (corrigé officiel) :

Si le patient est malade (insuffisance hépatique), il a toujours une élévation des transaminases → il n'existe pas de faux négatifs (FN = 0) → la sensibilité est de 100% (signe constant).

Ex 7QROC
(Données de l'Ex 1 : toxoplasmose.) Calculez le rapport de vraisemblance positif et négatif. Quelles conclusions sur la valeur informationnelle du test ?
Réponse type (corrigé officiel) :

RV+ = (54/58)/(1/10) = 9,3 → le test est 9,3 fois plus souvent positif chez les malades que chez les non-malades : test discriminant quand il est positif (pour affirmer le diagnostic).

RV− = (4/58)/(9/10) = 0,08 ≈ 1/12 → le test est 12 fois moins souvent négatif chez les malades : test discriminant pour exclure la maladie.

Ex 8QROC
Bandelette urinaire : sensibilité 90%, spécificité 60%. Une femme de 35 ans dysurique a une probabilité pré-test de 55% d'infection urinaire. Probabilité d'IU si la bandelette est positive ? de ne pas avoir d'IU si négative ? d'avoir une IU si négative ?
Réponse type (corrigé officiel) :
L'énoncé original écrit « une chance sur 55 » ; le corrigé officiel raisonne avec une prévalence de 55% (repris ici).

Sur 1000 patientes : prévalence 55% → 550 malades, 450 saines.

Se 90% → VP = 0,90 × 550 = 495 ; FN = 55. Sp 60% → VN = 0,60 × 450 = 270 ; FP = 180.

Total tests + = 495 + 180 = 675 ; total tests − = 55 + 270 = 325.

VPP = 495/675 = 73% → si bandelette positive, la probabilité d'IU passe de 55% à 73%.

VPN = 270/325 = 83% → probabilité de ne pas être malade si bandelette négative.

Probabilité d'IU si bandelette négative = 1 − VPN = 17%.

Ex 9QROC
(Bandelette urinaire, Se 90% / Sp 60%, pré-test 55%.) Que représente la probabilité pré-test ? Calculez RV+ et RV−. Avec le nomogramme de Fagan, probabilité post-test si bandelette positive ? si négative ?
Réponse type (corrigé officiel) :
Le corrigé écrit « pré-test 40% » par erreur ; tous les calculs utilisent bien la prévalence de 55% (cohérent avec l'Ex 8).

La probabilité pré-test (55%) = la prévalence de l'IU chez les femmes de 35 ans dysuriques (tableau clinique évocateur).

RV+ = (495/550)/(180/450) = 0,9/0,4 = 2,25. RV− = (55/550)/(270/450) = 0,1/0,6 = 0,17.

Fagan : droite partant de 55 (pré-test) via RV+ (2,25) → post-test ≈ 73% si bandelette positive ; via RV− (0,17) → post-test ≈ 17% si bandelette négative.

Ex 12QROC

Prévalence 40%EP +EP −Total
T +320120440

Réponse type (corrigé officiel) :

Pré-test 40% = prévalence de l'EP chez les sujets suspects.

RV+ = (320/400)/(120/600) = 0,8/0,2 = 4. RV− = (80/400)/(480/600) = 0,2/0,8 = 0,25.

Ex 13QROC
(D'après Junod.) Suspicion d'EP. Probabilité a priori : médecin 1 = 20%, médecin 2 = 50%, médecin 3 = 80%. Trois tests : D-dimères (positifs chez 98% des EP, et chez 55% des non-EP), CT-scan (Se 70% / Sp 90%), écho-doppler des MI (Se 50% / Sp 97%). Sensibilité/spécificité de chaque ? Intérêt diagnostique de chacun ?
Réponse type (corrigé officiel) :

D-dimères : Se = 98% ; Sp = 100 − 55 = 45%. CT-scan : Se 70% / Sp 90%. Écho-doppler : Se 50% / Sp 97%.

On construit, pour chaque test, un tableau de contingence dans les 3 situations de prévalence (20%, 50%, 80%) puis on lit la probabilité post-test (VPP si positif, 1−VPN si négatif).

Intérêt de chaque test :

• D-dimères — haute sensibilité (98%) : probabilité post-test basse en cas de test négatif → exclure l'EP.

• Écho-doppler — haute spécificité (97%) : confirmer l'EP si positif, mais ne permet pas de l'exclure si négatif.

• CT-scan — bonne Se (70%) et bonne Sp (90%) : moins bon que les D-dimères pour exclure, moins bon que l'écho-doppler pour confirmer.
