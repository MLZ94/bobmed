Partie 1 · Parasitologie

1 · Classification des parasites

Métazoaires

  • Arthropodes
  • Helminthes (vers)
    • Plats = plathelminthes : cestodes (Taenia, Bothriocéphale, Hymenolepis, Echinococcus granulosus/multilocularis) + trématodes (douves/Fasciola, bilharzioses/Schistosoma)
    • Ronds = némathelminthes : ovipares monoxènes (Ascaris, oxyure, trichocéphale, anguillule, ankylostome, Toxocara, Anisakis) ; vivipares hétéroxènes (trichine, filaires, Loa, onchocerque, filaire de Médine)

Protozoaires

  • Sporozoaires : Plasmodium, Toxoplasma gondii, coccidies intestinales
  • Rhizopodes : amibes, acanthamibes
  • Flagellés : Trypanosoma, Leishmania, Giardia, Trichomonas
🔑 Vers plats = trématodes (+ cestodes) ; vers ronds = nématodes. Ovipare = monoxène ; vivipare = hétéroxène.

Comportement alimentaire

Poisson : Anisakis, bothriocéphale, douves.

Pénétration transcutanée

Helminthes : larve L3 (anguillule, ankylostome), furcocercaire (schistosomes), vecteur (filaires, Loa, onchocerque). Acariens : sarcopte (gale). Insectes : diptères (myiases), puce chique (tungose).

Vecteurs (inoculation)

Protozoaires : Plasmodium, Leishmania, Trypanosoma.

HE autochtone

Modérée (< 1500/mm³) → scotch-test anal + EPS (± sérologies)Élevée (> 1500/mm³) → EPS + sérologies
Oxyure ; Taenia saginata ; échinococcose alvéolaire ; (kyste hydatique) ; (toxoplasmose) ; (hypodermose) Trichinellose ; larva migrans viscérale (toxocarose, anisakiase) ; fasciolose ; fissuration de kyste hydatique

HE tropicale

  • Nématodes intestinaux : tous (Ascaris, anguillule, ankylostome +++) → EPS + Baermann
  • Filaires : lymphatiques → leucoconcentration nocturne ; onchocercose → biopsie cutanée exsangue (BCE) ; loase → leucoconcentration diurne
  • Trématodes : bilharzioses → sérologie (invasion), EPS/EPU (état) ; distomatoses → sérologie + EPS
  • Cestodes : cysticercose → sérologie
🔑 Bilan « pour tout le monde » : EPS avec recherche d'anguillule + sérologies (bilharziose, distomatose, trichinellose, filariose, larva migrans, cysticercose) + leucoconcentration nocturne.
+ Afrique/Amérique → biopsie cutanée exsangue ; + Afrique/Arabie → EPU ; + bloc forestier congolais → leucoconcentration diurne.

HE + fièvre

Helminthiases en phase d'invasion : bilharziose aiguë, syndrome de Loeffler, trichinellose, distomatose ; fissuration de kyste hydatique ; isosporose, toxoplasmose.

4 · Toxoplasmose

Parasite & cycle

  • 3 formes : tachyzoïte (= forme végétative/trophozoïte) ; kyste (bradyzoïtes) ; oocyste (« œuf »).
  • Hôte définitif : chat / félidés (reproduction sexuée → oocystes). Homme et autres = hôtes intermédiaires (asexuée).
  • Cosmopolite ; prévalence France ≈ 31 % (2016).

Clinique

  • Bénigne chez l'immunocompétent (sauf souches virulentes, ex. Guyane).
  • Immunodéprimé : atteintes neurologiques et disséminées graves.
  • Fœtus (primo-infection gravidique) : toxoplasmose congénitale (≈ 200/an) → neuro/ophtalmo, fœtopathies graves.

Diagnostic = sérologique

  • Recherche IgG + IgM (obligatoire) ; mentionner réactif + valeurs seuils.
  • Séroconversion = sérologie négative → positive : c'est l'apparition des IgG qui l'affirme.
  • 2 sérologies comparables uniquement si même labo / même technique en parallèle.

Congénitale : risque vs gravité

T1T2T3
Transmission15 %30 %60 %
Gravitémaximale↓minime
Le risque de transmission ↑ avec le terme ; la gravité ↓ avec le terme.

Prévention

  • Femme enceinte séronégative : sérologie mensuelle (+ 3 sem post-accouchement) + échographies fœtales.
  • Si séroconversion gravidique : spiramycine, échographies, diagnostic anténatal (PCR liquide amniotique).
  • Mesures : pas de viande crue/peu cuite (ou congeler > 3 j), laver légumes/fruits, gants pour la terre/litière, éviter le chaton à la campagne.

Partie 2 · Mycologie

6 · Mycologie — généralités

Les champignons forment un règne autonome (à côté des animaux, plantes, protistes, procaryotes).

Thalle unicellulaire : levures

Candida +++ (≈ 80 % des infections fongiques), Cryptococcus, Pneumocystis, Malassezia. Multiplication par bourgeonnement (blastospores).

Thalle filamenteux

Moisissures (Aspergillus fumigatus), dermatophytes (Trichophyton), zygomycètes (Rhizopus). Filaments septés ou siphonnés.

Champignons dimorphiques

Levure à 37 °C (parasitaire) / filamenteux dans le milieu extérieur (saprophyte). « Exotiques » : Histoplasma, Blastomyces, Coccidioides, Sporothrix, Penicillium marneffei.

Origine

Endogène : Candida albicans (tractus digestif & uro-génital ; toujours pathogène sur peau/ongles). Exogène (majorité) : sol/air (Aspergillus, Cryptococcus, Histoplasma…).
🔑 Diagnostic : direct (examen direct +++ , culture, identification) vs indirect (Ac, Ag, ADN/PCR). L'identification ne dit pas si le champignon est impliqué dans la pathologie → discussion clinico-biologique +++ (la plupart sont commensaux).

8 · Champignons filamenteux

Dermatophytoses

  • Mycoses superficielles fréquentes (peau, phanères) ; dermatophytes kératinophiles/kératinolytiques, toujours pathogènes. Prévalence France 10-15 %.
  • Transmission : anthropophile +++ > zoophile > géophile ; indirecte (spores) > directe.
  • 4 genres : Microsporum, Trichophyton (T. rubrum +++), Epidermophyton, Nannizzia.
  • Clinique : dermatophyties circinées (peau glabre), des plis, teignes (cuir chevelu/barbe), onychomycoses. Prurit habituel. Examen mycologique obligatoire si atteinte des phanères.

Traitements des mycoses profondes

  • Candidose profonde / candidémie : échinocandine en 1re intention, réévaluation après identification + antifongigramme (14 j) ; retrait KT ; FO, écho cœur, hémocultures.
  • Pneumocystose : triméthoprime-sulfaméthoxazole (cotrimoxazole) curatif 3 sem ; prophylaxie I/II si ID.
  • Cryptococcose neuroméningée / disséminée : amphotéricine B + 5-FC 2 sem, puis fluconazole 8 sem.
  • Aspergillose invasive : voriconazole ou isavuconazole (6-12 sem, dosages des azolés).
